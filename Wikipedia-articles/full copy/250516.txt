{ Mikael Nalbandian (1829-1866) } { 1859 Իտալացի աղջկա երգը The Song of the Italian girl in Armenian Wikisource } { Barsegh Kanachyan (1885-1967) } { 1991 } { Mer Hayrenik instrumental.ogg } { Mer Hayrenik (Instrumental) } Mer Hayrenik ( [ Մեր Հայրենիք ]; translated as Our Fatherland) is the national anthem of the Republic of Armenia. Adopted on July 1, 1991, it was also the national anthem of the Democratic Republic of Armenia (1918–1920), the first modern Armenian state. The lyrics of the anthem are adapted from a version of The Song of an Italian girl ( [ Իտալացի աղջկա երգը ], written in 1859) by Mikael Nalbandian (1829–1866). Later set to music by composer Barsegh Kanachyan (1885–1967).
