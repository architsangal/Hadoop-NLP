This is a list of mail servers: mail transfer agents, mail delivery agent s, and other computer software which provide e-mail services. [ Product statistics ] According to one survey, sendmail, Microsoft Exchange Server, Postfix, and Exim together control over 85% of market share for SMTP service. { Security Space Mail (MX) Server Survey, Aug 1, 2010. } Another survey suggests a more balanced distribution, though it included hosted e-mail services such as Postini. { MailChannels Mail Server Survey, 2006 } These surveys are rather difficult to produce, [ http://www.oreillynet.com/lpt/a/6849 Fingerprinting the World&apos;s Mail Servers Ken Simpson Stas Bekman 2007 O&apos;Reilly Media October 23, 2010 ] hence it is no surprise if they disagree.
