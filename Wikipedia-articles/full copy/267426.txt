[ נַחְף ] { Náḥep } { نحف } { lc } { Nahef } { Naḥf } { north } { 7,800 } { 2011 } { 6077 } { Israel northwest } { 250 } { 32 } { 55 } { 53.94 } { 35 } { 19 } { 10.51 } Nahf ( [ نحف ], Naḥf or Nahef; [ נַחְף ] ) { Palmer, 1881, p.64, p.92 } is an Arab town in the North District Israel. It is located in the between the Lower and Upper Galilee, about [ 23 km mi ] east of Acre. In 2011, the population was 7,800. { Integrating Arabs in the Workforce, [[Jerusalem Post ]] } [ History ] The village is of considerable age, and it contains Roman and medieval remains. It was known as &quot; Nef &quot; in Crusader period. { Pringle, 1997, p.114 } { Petersen, 2002, p. 239 } { Conder and Kitchener, 1881, I:255. Cited in Petersen, 2002, p. 239 } In 1250 it was mentioned together with Beit Jenn, Jalun and other places in the neighborhood as being among the Casales belonging to the Knights of Teutonic Order. { Conder and Kitchener, 1881, p.208 } A map from 1799 by Pierre Jacotin showed the place, named as &quot; Nafeh &quot;. { Karmon, 1960, p.166, Cited in Petersen, 2002, p. 239 } When Victor Guérin visited Nahf in the 1870s, he described the village as containing 400 Muslim s and some Greek Orthodox families, { Guérin, 1880, p.451 - p.452 Cited partially in Conder and Kitchener, 1881, I:255. Translation in Petersen, 2002, p. 239 } while in 1881 Nafh was described as being built of stone, containing 200 Muslims, with olive s and arable land. { Conder and Kitchener, 1881, p.203 } Nahf was captured by Israel on July 18, 1948 during Operation Dekel led by the Sheva (Seventh) Brigade. Its defenders included the town&apos;s local militia as well as Arab Liberation Army volunteers. The town was left intact and most residents did not flee their homes.In and around Nahf, there are a number of archeological remains dating from the Middle Ages, including mosaics and a cemetery. In a nearby location lies the shrines of Muslim leaders of Sheik Muhammad Rabiah and Sheik Mahmud who fought against the Crusaders. The Auba cave, which dates from the time of the Assyrians is also located here. { Welcome To Nahf } [ Notable structures ] The largest medieval structure in the village is a ca. 10 m. long wall, made of large drafted blocks with a rubble core, which may be of Crusader origin. Maqam of Shaykh Rabi is located on a steep hill above the village, surrounded by a cemetery. It is a dome d rectangular building, with an entrance from the east. A deep mihrab is located inside, in the middle of the south side. By the north wall is the elongated cenotaph of Shaykh Rabi.
