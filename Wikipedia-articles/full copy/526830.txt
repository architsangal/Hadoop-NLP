This is a list of cruiser s of the Royal Navy of the United Kingdom from 1877 (when the category was created by amalgamating the two previous categories of frigate and corvette ) until the last cruiser was deleted more than a century later. There are no longer any cruisers in the Royal Navy. [ First class cruisers ] Armoured cruiser s were protected by a belt of side armour and an armoured deck. In the Royal Navy this classification was not actually used, the term first class cruiser being used instead for both armoured cruisers and large protected cruiser s. Thus, the first class cruisers built between the &apos;&apos;Orlando&apos;&apos; class (1886) and the &apos;&apos;Cressy&apos;&apos; class (1897) were, strictly speaking, protected cruisers as they lacked an armoured belt. This makes the Arethusas the first &quot; true example &quot; of the warship that came to be recognised as the light cruiser.
